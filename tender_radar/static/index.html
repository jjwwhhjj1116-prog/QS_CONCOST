나라장터·LH API 승인 순서
    
        
  1. 공공데이터포털에서 나라장터 입찰공고정보서비스와 LH 입찰공고정보를 각각 엽니다.
    2. 
        
  2. 두 서비스 모두 활용신청을 누르고 활용 목적은 웹사이트 개발 또는 업무용 정보수집으로 작성합니다.
    3. 
        
  3. 신청을 제출한 뒤 마이페이지 → 데이터활용 → Open API → 개발계정으로 이동합니다.
    4. 
        
  4. 승인된 서비스의 상세보기에서 일반 인증키(Decoding)를 복사합니다. 키 하나를 같이 쓰지만 서비스별 활용승인은 각각 필요합니다.
    5. 
        
  5. 위 입력창에 붙여넣고 API 키 저장 → 지금 공고 수집을 누릅니다.
